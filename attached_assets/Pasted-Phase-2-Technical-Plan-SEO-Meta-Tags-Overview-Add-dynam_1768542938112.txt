Phase 2 Technical Plan: SEO Meta Tags
Overview
Add dynamic meta tags to /search/:slug/:listId pages for better search rankings and social sharing.

Implementation Approach
Option A: React Helmet (Recommended)
You likely already have react-helmet or react-helmet-async in your project. Add meta tags directly in the page component.
File: client/src/pages/home.tsx (or wherever the search route renders)
tsximport { Helmet } from 'react-helmet';

// Inside component, after fetching list data:
<Helmet>
  <title>{list.prompt} | 5Ducks</title>
  <meta name="description" content={`${list.resultCount} companies found: ${list.prompt}`} />
  
  {/* Open Graph */}
  <meta property="og:title" content={`${list.prompt} | 5Ducks`} />
  <meta property="og:description" content={`${list.resultCount} companies found for lead generation`} />
  <meta property="og:type" content="website" />
  <meta property="og:url" content={`https://5ducks.ai/search/${slug}/${listId}`} />
  
  {/* Twitter */}
  <meta name="twitter:card" content="summary" />
  <meta name="twitter:title" content={`${list.prompt} | 5Ducks`} />
  <meta name="twitter:description" content={`${list.resultCount} companies found`} />
</Helmet>

Structured Data (JSON-LD)
Add to the same component:
tsx<Helmet>
  <script type="application/ld+json">
    {JSON.stringify({
      "@context": "https://schema.org",
      "@type": "CollectionPage",
      "name": list.prompt,
      "description": `${list.resultCount} companies matching: ${list.prompt}`,
      "url": `https://5ducks.ai/search/${slug}/${listId}`,
      "numberOfItems": list.resultCount,
      "provider": {
        "@type": "Organization",
        "name": "5Ducks",
        "url": "https://5ducks.ai"
      }
    })}
  </script>
</Helmet>

Implementation Steps

Check if react-helmet exists in project (2 min)
If not, install it: npm install react-helmet (1 min)
Add Helmet block to home.tsx inside the search route condition (15 min)
Test by inspecting page source on a search URL (5 min)

Total: ~25 minutes

Fallback Note
If the page hasn't loaded list data yet, use sensible defaults:
tsxconst pageTitle = list?.prompt 
  ? `${list.prompt} | 5Ducks` 
  : 'Search Results | 5Ducks';
This prevents empty meta tags during loading.
